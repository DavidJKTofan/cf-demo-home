Description

The Cloudflare WAF is a Web Application Firewall that protects your HTTP/S web applications from common and new vulnerabilities. It mainly consist of the following components:

  • Traffic Detections
  • Custom Rules
  • Rate Limiting Rules
  • WAF Managed Rules
  • Additional Tools
