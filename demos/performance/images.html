Description

Cloudflare Images is an end-to-end image management solution that lets you store, resize, optimize and deliver images in a fast and secure way.

Key Features:

  • Image Upload: Direct creator uploads with one-time upload URLs
  • Image Variants: Create preset variants for different use cases
  • Flexible Transformations: Enable flexible variants to apply more customizable transformation parameters, similar to Image Transformation.
  • Privacy Controls: Serve private images using signed URL tokens

Alternatively, for other hosting options, check out Cloudflare's Developer Platform.

Image URL Format

Standard Domain:

https://imagedelivery.net/<ACCOUNT_HASH>/<IMAGE_ID>/<VARIANT_NAME>

Custom Domain:

https://example.com/cdn-cgi/imagedelivery/<ACCOUNT_HASH>/<IMAGE_ID>/<VARIANT_NAME>
