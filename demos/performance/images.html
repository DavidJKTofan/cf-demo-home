Description

Cloudflare Images is a managed-images hosting service that automatically optimizes images on-demand. Some relevant features are:

  • accept user-uploaded images with a one-time upload URL.
  • Create variants of images on-demand.
  • Enable flexible variants to apply more customizable transformation parameters, similar to the Image Transformation service.
  • Serve private imagesby using signed URL tokens.

Alternatively, for other hosting options, check out Cloudflare's Developer Platform.
