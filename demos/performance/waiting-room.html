Waiting Room

Description

Explore how Cloudflare manages surges in website traffic with virtual waiting rooms and different queueing methods.

There are several relevant features:

  • Customize the Waiting Room design, which also supports multiple-languages
  • Enable SEO Crawler Bypassing
  • Schedule events
  • Send bots to an infinite queue

This solution is part of the preparing for surges or spikes in web traffic guide, especially useful during seasonal events, such as Black Friday or Christmas.


After an event ends, the reject queueing method can be used to block traffic from reaching the application, displaying a custom HTML template instead. This template can also be updated or patched to signal that the event is over.
