DNS Image Transformation

Description

There are two main methods to transform images:

  • via URL - usually also used with srcset
  • via Workers

The Workers method allows for more programmatic customization, as well as more complex transformations.

The URL method is simpler to implement, such as using width=auto to automatically resize the image based on the available browser and device information.
