R2 Object Storage

Description

Cloudflare R2 is a low-cost, high-performance object storage solution that helps you store and serve any type of data. The two main ways of interacting with R2 are via the S3 API or Workers API, benefiting from zero egress fees and making it easy to integrate with existing applications and workflows.

Some relevant features:

  • Easy data migration
  • Object lifecycles
  • Public buckets (they are private by default, but can also be configued with Access control)
  • Presigned URLs
  • Jurisdictional Restrictions
  • Metrics & Analytics
  • Audit Logs
